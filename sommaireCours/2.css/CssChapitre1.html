Chapitre 1 : Inline, interne et externe

Introduction aux différentes façons d’inclure du CSS

Le CSS peut être ajouté à une page web de trois manières principales :

  • Inline : directement dans la balise HTML via l’attribut style.
  • Interne : dans une balise <style> au sein du fichier HTML.
  • Externe : via un fichier CSS séparé référencé avec <link>.

Styles inline

Les styles inline s’appliquent directement à un élément HTML :

<h2 style="color: blue; font-size: 20px;">Titre bleu</h2>

Avantages : rapide à utiliser, pratique pour un style ponctuel.
Inconvénients : difficile à maintenir, ne sépare pas la structure du style.

Styles internes

Les styles internes sont définis dans une balise <style> dans le <head> du document :

<style>
    h2 {
        color: green;
        font-size: 24px;
    }
    </style>

Avantages : utile pour des styles propres à une seule page.
Inconvénients : styles mélangés avec le HTML, pas réutilisables sur plusieurs pages.

Styles externes

Les styles externes sont placés dans un fichier CSS séparé, puis reliés avec :

<link rel="stylesheet" href="style.css">

Avantages : styles centralisés, maintenance simplifiée, partage facile entre plusieurs pages.
Inconvénients : nécessite une requête supplémentaire, mais largement compensé par la mise en cache.
