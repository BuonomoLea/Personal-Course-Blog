Chapitre 3 : HTML — Balises inline et block

Balises inline : éléments en ligne

Les balises inline ne provoquent pas de retour à la ligne. Elles s’intègrent dans le flux du texte.

  • <span> : Conteneur de texte en ligne.
  • <a> : Lien hypertexte.
  • <strong> : Texte à forte importance (souvent en gras).
  • <img> : Image insérée dans le texte.

Balises block : éléments de bloc

Les balises block s’affichent sur une nouvelle ligne et prennent toute la largeur disponible.

  • <div> : Conteneur générique.
  • <p> : Paragraphe de texte.
  • <h1>, <h2>, <h3>, etc. : Titres hiérarchiques.
  • <ul>, <ol>, <li> : Listes (non numérotées, numérotées et leurs éléments).

Comparaison visuelle

Voici un exemple de balise block :

<p>Ceci est un paragraphe (block).</p>

Et un exemple de balise inline :

<span>Ceci est un span (inline).</span>

Erreurs courantes à éviter

  • Placer une balise block à l’intérieur d’une balise inline (non valide).
  • Forcer inutilement un comportement (ex. : inline en block ou inversement via CSS).
  • Ignorer leur influence sur la structure et l’accessibilité.
