Chapitre 7 : Les formulaires

Création de l’HTML


    <!--Formulaire classic-->

    <form action="/submit" method="post">

        <label for="name">Nom:</label><br>
        <input type="text" id="name" name="name" required minlength="5"><br>

        <label for="email">Email:</label><br>
        <input type="email" id="email" name="email" required minlength="5"><br>

        <label for="password">Mot de passe:</label><br>
        <input type="password" id="password" name="password" required minlength="5"><br>

        <!--Ici la div permet d’englober et disposer en ligne avec display: flex;-->

        <!--Radio-->
        <div id="containerRadio">
            <label>Genre:</label><br>
            <input type="radio" id="male" name="gender" value="male">
            <label for="male">Homme</label>
            <input type="radio" id="female" name="gender" value="female">
            <label for="female">Femme</label><br>
        </div>

        <!--Case à cocher-->
        <div id="containerAge">
            <label for="age">Age:</label><br>
            <input type="checkbox" id="age" name="age" value="mineur">
            <label id="labelAge" for="age">Je suis mineur</label><br>
        </div>

        <!--Liste déroulante-->
        <label for="majeur">Majeur:</label>
        <select id="majeur" name="majeur">
        <option value="18-30">18-30ans</option>
        <option value="30-45">30-45ans</option>
        <option value="45-60">45-60ans</option>
        <option value="60+">60+</option>
        </select><br>

        <!--Envois du formulaire-->
        <input type="submit" value="Soumettre">

    </form>

Condition de validation pour l’envoi en HTML

Required : Le champs doit obligatoirement être rempli pour l’envoie.

<input type="text" id="name" name="name" required>

Minlength : Le nombre minimum de caractère à entrer dans le champ.

Maxlength : Le nombre maximum de caractère à entrer dans le champ.

<input type="text" id="name" name="name" minlength="2" maxlength="10">

Pattern : défini une expression régulière (regex) à laquelle la valeur du champ de saisie doit correspondre


    <!--Ex pour un code postal français, on définit “uniquement 5 chiffres” = "\d{5}"-->
    <input type="text" id="postcode" name="postcode" pattern="\d{5}">

Ajouter du Css aux éléments invalides


    input:invalid {
    border: 2px solid red;
    background-color: #ffdddd;
    box-shadow: inset 0 0 5px red;
    color: red;
    }
    input:invalid::after {
    content: " ❌"; /* ajoute une croix rouge après l'élément invalide */
    }
    input:invalid {
    border: 2px solid;
    border-image: linear-gradient(to right, red, orange) 1;
    }

Explication du fonctionnement du form

Les attribut sont indispensables au fonctionnement :

method : indique par quel moyen les données vont être envoyées, les méthodes les plus utiliser sont get et post

get : obtenir

post : envoyer

action : indique l'adresse de la page ou du programme qui va traiter les informations

<form method="get" action=""></form>

Input :

Champs de texte avec input :

<input type="text">

Pour un nom ou prénom par exemple

Attribut name :

Ce nom n'apparaît pas sur la page, il permettra plus tard d'identifier le champ d'où viennent les informations.

<input type="text" name="prenom">

Libellé avec label :

Pour indiquer ce qu'il faut écrire dans la zone de texte par l'utilisateur on ajoute un libellé

C'est un champ de texte prérempli avec une indication pour guider le visiteur du site sur ce qu'il doit écrire

Il faut ensuite lier le label à a zone de texte pour cela on doit donner un nom à la zone de texte avec l'attribut id

Pour lier le label au champ, il faut lui donner un attribut for qui a la même valeur que l'id du champ :

<label for="prenom">Votre prénom</label> : <input type="text" name="prenom" id="prenom">

Liste des types pour input :


    Un email : <input type="email">
    Une url : <input type="url">
    Un numéro de tel : <input type="tel">
    Un nombre entier : <input type="number">
    Un curseur/slider : <input type="range">
    Une date : <input type="date">
    Pour l'heure : <input type="time">
    La semaine : <input type="week">
    Le mois : <input type="month">
    Pour la date et l'heure : <input type="datetime"> avec gestion du décalage horaire
    Pour la date et l'heure : <input type="datetime-local"> sans gestion du décalage horaire
    Une recherche : <input type="search">

Autres attributs pour input :

  • Size : agrandir le champ
  • maxlength : limiter le nombre de caractères que l'on peut saisir
  • value : préremplir le champ avec une valeur par défaut
  • placeholder : donner une indication sur le contenu du champ
  • min / max : valeur minimale/maximale autorisée
  • step : exemple : 0, 2, 4, 6…

Boutons de choix :

Cases à cocher :

<input type="checkbox" name="choixName" >

Avec attribut checked :

<input type="checkbox" name="choix" checked>

Boutons radio :

Un seul choix possible parmi plusieurs options :


    <form method="get" action="">
        <input type="radio" name="age" value="moins15" id="moins15">
        <label for="moins15">Moins de 15ans</label>
        <input type="radio" name="age" value="medium15-25" id="medium15-25">
        <label for="medium15-25">15-25 ans</label>
        <input type="radio" name="age" value="medium25-40" id="medium25-40">
        <label for="medium25-40">25-40 ans</label>
        <input type="radio" name="age" value="plus40" id="plus40">
        <label for="plus40">Plus de 40 ans</label>
    </form>

Regroupement avec fieldset et legend


    <fieldset>
    <legend>Informations personnelles</legend>
    <label for="nom">Nom :</label>
    <input type="text" id="nom" name="nom">
    <label for="email">Email :</label>
    <input type="email" id="email" name="email">
    </fieldset>

Attributs HTML pour la validation de base

  • required : Rend le champ obligatoire
  • minlength / maxlength : Limite la taille du texte saisi
  • pattern : Définition d'une expression régulière à respecter
  • type : Champ validé selon son type
  • min / max : Valeurs numériques minimales/maximales
  • step : Pas d'incrément
  • placeholder : Texte indicatif
  • readonly : Visible mais non modifiable
  • disabled : Grisé et ignoré
  • autofocus : Place le curseur automatiquement

    <input type="email" required placeholder="exemple@domaine.fr">
    <input type="number" min="1" max="10" step="1">
