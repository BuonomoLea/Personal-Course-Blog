Chapitre 2 : Les sélecteurs

Introduction aux sélecteurs CSS

Les sélecteurs permettent de cibler les éléments HTML auxquels appliquer des règles CSS.

Sélecteurs simples

  • p : sélectionne tous les éléments <p>.
  • .btn : sélectionne tous les éléments avec la classe btn.
  • #header : sélectionne l’élément avec l’ID header.
p {
    color: blue;
}
.btn {
    font-weight: bold;
}
#header {
    background-color: #eee;
}

Combinaisons & descendants

  • ul > li : sélectionne les li enfants directs de ul.
  • nav a : sélectionne tous les liens a descendants d’un nav.
  • h2 + p : sélectionne le p immédiatement après un h2.

Sélecteurs d’attribut

  • input[required] : éléments input avec l’attribut required.
  • a[href="page.html"] : liens dont l’attribut href vaut exactement page.html.
  • img[src^="https"] : images dont l’attribut src commence par https.
  • img[src$=".png"] : images dont le src se termine par .png.

Pseudo-classes

Les pseudo-classes ciblent des états particuliers des éléments :

  • a:hover : lien au survol
  • input:focus : champ de formulaire actif
  • li:first-child : premier enfant
  • :nth-child(n), :first-of-type, etc.

Pseudo-éléments

  • ::before
  • ::after
  • ::first-letter
  • ::selection
