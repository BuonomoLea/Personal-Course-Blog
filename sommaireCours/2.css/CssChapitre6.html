Chapitre 6 : Box Model

Qu’est-ce que le Box Model ?

Le modèle de boîte décrit la manière dont chaque élément HTML est modélisé dans le navigateur :

  • Content – Le contenu réel (texte, image, etc.)
  • Padding – L’espace entre le contenu et la bordure
  • Border – La bordure entourant l’élément
  • Margin – L’espace entre l’élément et les autres éléments

Illustration du modèle

+---------------------------+
    |        margin            |
    |  +--------------------+  |
    |  |     border         |  |
    |  |  +-------------+   |  |
    |  |  |  padding    |   |  |
    |  |  | +--------+  |   |  |
    |  |  | | content|  |   |  |
    |  |  | +--------+  |   |  |
    |  |  +-------------+   |  |
    |  +--------------------+  |
    +---------------------------+

Exemple CSS

.box {
    padding: 20px;
    border: 2px solid black;
    margin: 30px;
    }

Propriété box-sizing

Par défaut, la taille d’un élément ne comprend que le contenu. En utilisant :

* {
    box-sizing: border-box;
    }

Le padding et la bordure sont inclus dans la largeur/hauteur définie.
