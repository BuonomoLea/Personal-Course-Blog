Terminal :

Manipulation des fichiers

Ces commandes vous permettent de copier, déplacer, ou afficher vos fichiers.

touch NomDuFichier : Crée un fichier vide nommé NomDuFichier.

Exemple : touch fichier_nouveau.txt.

Utilité : Préparer un fichier pour y travailler.

cp Source Destination : Copie un fichier ou un dossier vers une autre destination.

Exemple : cp document.txt copie_document.txt ou cp -r DossierSource/ DossierDestination/.

Utilité : Dupliquer des fichiers ou dossiers.

mv Source Destination : Déplace ou renomme un fichier.

Exemple : mv fichier1.txt dossier1/ (déplace le fichier dans un dossier) ou mv ancien_nom.txt nouveau_nom.txt (renomme le fichier).

cat NomDuFichier : Affiche le contenu d’un fichier texte dans le terminal.

Exemple : cat notes.txt.

Utilité : Lire rapidement un fichier sans l’ouvrir dans un éditeur.
