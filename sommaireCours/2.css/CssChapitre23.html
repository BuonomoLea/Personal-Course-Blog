Chapitre 23 : Mini design system & utilitaires

Qu’est-ce qu’un mini design system ?

Un mini design system regroupe des composants CSS réutilisables, facilitant la cohérence et la maintenance du style sur un projet.

Utilitaires CSS courants

Les classes utilitaires permettent d’appliquer rapidement des styles simples comme les marges, paddings, alignements ou couleurs.

Avantages d’un design system

  • Gain de temps en développement
  • Uniformité visuelle
  • Facilité de maintenance

1. Fichier variables.css

Définir toutes les couleurs, tailles, espacements et polices sous forme de variables CSS permet de centraliser le thème et de faciliter la maintenance.

:root {
            /* couleurs */
            --color-primary:   #007bff;
            --color-secondary: #6c757d;
            --color-success:   #28a745;
            --color-danger:    #dc3545;
            --color-warning:   #ffc107;
            --color-info:      #17a2b8;
            --color-light:     #f8f9fa;
            --color-dark:      #343a40;

            /* typographie */
            --font-family-sans:   'Helvetica Neue', Arial, sans-serif;
            --font-size-base:     1rem;
            --line-height-base:   1.5;

            /* espacements */
            --spacer: 1rem;
            --spacer-sm: .5rem;
            --spacer-lg: 2rem;

            /* rayons de bordure */
            --radius: .25rem;
            --radius-lg: .5rem;
            }

2. Reset CSS

Réinitialiser les marges, paddings et styles natifs des navigateurs pour partir sur une base uniforme et cohérente.

*, *::before, *::after {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
            }
            html {
            font-family: var(--font-family-sans);
            font-size: var(--font-size-base);
            line-height: var(--line-height-base);
            scroll-behavior: smooth;
            }
            body {
            background: var(--color-light);
            color: var(--color-dark);
            }
            /* Médias */
            img, picture, video, canvas, svg {
            display: block;
            max-width: 100%;
            height: auto;
            }
            /* Liens et boutons */
            a {
            text-decoration: none;
            color: inherit;
            }
            button {
            background: none;
            border: none;
            cursor: pointer;
            font: inherit;
            }
            ul {
            list-style: none;
            }

3. Utilitaires CSS réutilisables

Ces classes permettent de gérer rapidement la visibilité, les espacements ou la présentation de base de certains éléments.

/* cacher visuellement mais accessible */
            .visually-hidden, .sr-only {
            position: absolute !important;
            width: 1px;
            height: 1px;
            padding: 0;
            margin: -1px;
            overflow: hidden;
            clip: rect(0, 0, 0, 0);
            white-space: nowrap;
            border: 0;
            }

            /* bouton standard */
            .btn {
            display: inline-block;
            padding: var(--spacer-sm) var(--spacer);
            font-size: var(--font-size-base);
            line-height: var(--line-height-base);
            border-radius: var(--radius);
            background: var(--color-primary);
            color: #fff;
            text-align: center;
            transition: background .2s;
            }
            .btn:hover, .btn:focus {
            opacity: .9;
            }

            /* conteneur centré */
            .container {
            width: 100%;
            margin-left: auto;
            margin-right: auto;
            padding-left: var(--spacer);
            padding-right: var(--spacer);
            }
            @media (min-width: 576px) { .container { max-width: 540px; } }
            @media (min-width: 768px) { .container { max-width: 720px; } }
            @media (min-width: 992px) { .container { max-width: 960px; } }
            @media (min-width: 1200px){ .container { max-width: 1140px; } }

4. Architecture CSS par composants

Organiser les feuilles de style CSS en modules spécialisés simplifie la gestion du code à mesure que le projet grandit.

  • variables.css
  • reset.css
  • base.css (typographie, formulaires, etc.)
  • layout.css (grilles, conteneurs, header/footer)
  • components/ :
    • _buttons.css
    • _cards.css
    • _modals.css
    • _accordions.css
    • _tooltips.css
  • utilities.css

Exemple de fichier principal main.css :

@import 'variables.css';
            @import 'reset.css';
            @import 'base.css';
            @import 'layout.css';
            @import 'components/buttons.css';
            @import 'components/cards.css';
            @import 'components/modals.css';
            @import 'components/accordions.css';
            @import 'components/tooltips.css';
            @import 'utilities.css';
