Curl :

Curl

Pour vérifier les connexions ou télécharger des fichiers.

ping AdresseIP : Vérifie si une machine ou un serveur est joignable.

Exemple : ping 8.8.8.8 (teste la connexion au serveur DNS de Google).

Utilité : Diagnostiquer des problèmes réseau.

curl -O URL : Télécharge un fichier depuis une URL.

Exemple : curl -O http://example.com/fichier.txt.

Utilité : Obtenir des fichiers depuis Internet directement dans le terminal.

curl -O https://www.example.com/images/photo.jpg

Ce que ça fait : Télécharge une image nommée photo.jpg depuis le site web dans le répertoire courant.

Pour extraire des informations spécifiques d'une page web (comme des liens ou des images), vous pouvez combiner curl avec des outils comme grep ou awk pour analyser le contenu HTML téléchargé.

curl http://example.com | grep "<img>"

Curl accède directement à une ressource en ligne via une URL pour télécharger les éléments spécifiés. Il est puissant et peut aller bien au-delà du simple téléchargement de fichiers : il permet d'interagir directement avec des serveurs web, de simuler des navigateurs, et même d’extraire des données à des fins spécifiques.

Télécharger une page web en HTML

curl -O https://example.com/index.html

Ce que ça fait : Télécharge le fichier index.html de la page web.

Utilité : Si vous travaillez sur un projet où vous devez analyser ou comprendre le code source d’un site web.

curl https://example.com | grep "href"

Ce que ça fait : Télécharge temporairement la page HTML, affiche uniquement les lignes contenant des liens <a href="...">.

Utilité : Extraire facilement tous les liens d’une page web.

Envoyer des requêtes spécifiques

curl peut simuler différents types de requêtes HTTP, telles que GET (par défaut), POST, ou PUT.

Exemple : Effectuer une requête POST

curl -X POST -d "username=test&password=1234" https://example.com/login

Ce que ça fait : Envoie des données username et password en tant que formulaire à une URL.

Utilité : Tester des formulaires web ou interagir avec des APIs.

Simuler un navigateur

Certains serveurs attendent qu’un navigateur particulier effectue la requête. Avec curl, vous pouvez définir un "User-Agent" pour simuler cela.

Exemple : Simuler Google Chrome

curl -A "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/91.0.4472.124 Safari/537.36" https://example.com

Ce que ça fait : Simule une requête effectuée depuis Google Chrome.

Utilité : Accéder à des sites restreints aux navigateurs spécifiques ou contourner des limitations légères.

Suivre les redirections

Exemple : Suivre une URL redirigée

curl -L -O https://bit.ly/fichier

Ce que ça fait : Suit automatiquement les redirections (exemple : des liens raccourcis comme bit.ly).

Interagir avec une API

De nombreuses APIs utilisent curl pour envoyer et récupérer des données. Par exemple, une API REST peut répondre avec du JSON.

Exemple : Récupérer des données JSON d’une API

curl https://api.example.com/data.json

Ce que ça fait : Télécharge un fichier JSON contenant des données depuis l’API.

Utilité : Extraire des données pour un traitement automatisé.

Exemple : Utiliser une clé d'authentification avec une API

curl -H "Authorization: Bearer VOTRE_CLE" https://api.example.com/secure-data

Ce que ça fait : Ajoute un en-tête d’autorisation pour accéder à des données sécurisées.

Utilité : Interagir avec des services web nécessitant des permissions spéciales.

Télécharger un fichier avec une limite de vitesse

curl --limit-rate 100k -O https://example.com/largefile.iso

Ce que ça fait : Limite la vitesse de téléchargement à 100 kB/s.

Utilité : Tester la gestion de bande passante ou réduire l’utilisation réseau.

Sauvegarder les en-têtes de réponse

curl -I https://example.com

Ce que ça fait : Affiche uniquement les en-têtes de la réponse HTTP (exemple : type de contenu, statut, cookies, etc.).

Utilité : Obtenir des informations sur le serveur ou la ressource sans télécharger le contenu complet.

Télécharger des fichiers sécurisés

Si un site utilise une connexion HTTPS sécurisée avec des certificats, curl gère cela automatiquement. Cependant, vous pouvez spécifier vos propres certificats si nécessaire.

Exemple : Télécharger avec un certificat spécifique

curl --cert certificat.pem https://secure.example.com

Note : curl seul ne permet pas de télécharger des vidéos depuis YouTube, car YouTube utilise des mécanismes complexes pour diffuser et protéger ses contenus (même lorsqu'ils sont libres de droits). Les URLs de YouTube ne pointent pas directement vers des fichiers téléchargeables, mais vers des flux vidéo qui nécessitent une interprétation spécifique.

Cependant, pour ce type de tâche, des outils spécialisés comme youtube-dl ou son successeur yt-dlp sont recommandés. Ils permettent de télécharger des vidéos ou des audios de YouTube (ou d'autres plateformes similaires), tout en respectant les droits d'utilisation selon les conditions de service.
