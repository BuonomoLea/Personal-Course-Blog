Slider

<!-- Structure HTML minimale -->
        <div class="slider">
        <div class="slider__track">
            <div class="slider__slide">Slide 1</div>
            <div class="slider__slide">Slide 2</div>
            <div class="slider__slide">Slide 3</div>
        </div>
        <button class="slider__button slider__button--prev" aria-label="Précédent">‹</button>
        <button class="slider__button slider__button--next" aria-label="Suivant">›</button>
        <div class="slider__dots">
            <button class="slider__dot" aria-label="Aller à la slide 1"></button>
            <button class="slider__dot" aria-label="Aller à la slide 2"></button>
            <button class="slider__dot" aria-label="Aller à la slide 3"></button>
        </div>
        </div>
        
/* Slider Component */
        .slider {
        position: relative;
        overflow: hidden;
        }

        .slider__track {
        display: flex;
        transition: transform 0.4s cubic-bezier(.25,.8,.25,1);
        will-change: transform;
        }

        .slider__slide {
        flex: 0 0 100%;
        display: flex;
        align-items: center;
        justify-content: center;
        }

        .slider__button {
        position: absolute;
        top: 50%;
        transform: translateY(-50%);
        width: 2rem;
        height: 2rem;
        background: #fff;
        border: 1px solid #ccc;
        border-radius: 50%;
        cursor: pointer;
        font-size: 1.25rem;
        line-height: 1;
        display: flex;
        align-items: center;
        justify-content: center;
        transition: background 0.2s;
        }

        .slider__button:hover {
        background: #f0f0f0;
        }

        .slider__button--prev {
        left: 1rem;
        }

        .slider__button--next {
        right: 1rem;
        }

        .slider__dots {
        position: absolute;
        bottom: 1rem;
        left: 50%;
        transform: translateX(-50%);
        display: flex;
        gap: 0.5rem;
        }

        .slider__dot {
        width: 0.75rem;
        height: 0.75rem;
        background: #ccc;
        border: none;
        border-radius: 50%;
        padding: 0;
        cursor: pointer;
        transition: background 0.2s;
        }

        .slider__dot.is-active {
        background: #007acc;
        }

Script

Calculer transform: translateX(-n*100%) sur .slider__track

Ajouter/retirer la classe .is-active sur les .slider__dot

Gérer les clics sur Prev/Next et les dots.
