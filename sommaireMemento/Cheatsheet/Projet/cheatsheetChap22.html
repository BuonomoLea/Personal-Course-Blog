Terminal :

Navigation dans le système de fichiers

Ces commandes permettent de se déplacer et d’interagir avec la structure des fichiers et dossiers.

ls : Liste les fichiers et dossiers dans le répertoire courant.

Exemple : ls ou ls -l (affiche des détails supplémentaires comme les droits d’accès).

Utilité : Voir le contenu d’un dossier.

cd NomDuDossier : Change le répertoire de travail courant pour NomDuDossier.

Exemple : cd Projet.

Utilité : Naviguer dans vos dossiers.

cd .. : Revient au répertoire parent.

Exemple : Si vous êtes dans Documents, cd .. vous ramène dans le répertoire qui contient Documents.

pwd : Affiche le chemin complet du répertoire courant.

Exemple : /home/utilisateur/Projet.

Utilité : Savoir exactement où vous vous trouvez dans l’arborescence des dossiers.

mkdir NomDuDossier : Crée un nouveau dossier nommé NomDuDossier.

Exemple : mkdir Test.

Utilité : Organiser vos fichiers en créant des répertoires.

rm NomDuFichier : Supprime un fichier.

Exemple : rm ancien_fichier.txt.

Attention : La suppression est irréversible.

rm -r NomDuDossier : Supprime un dossier et tout son contenu (récursif).

Exemple : rm -r DossierInutile
