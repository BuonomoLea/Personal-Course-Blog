Démarrer un projet

1. Créer un dossier de projet

On créer un dossier avec le nom du projet dans le dossier :

nameProject

2. Créer les fichiers de base

Dans VSCode on ouvre ce dossier pour y créer :

  • Une page index.html
  • Une page style.css
  • Une page script.js
  • Une page server.js → pour le côté serveur

3. Lier les fichiers dans la page HTML

Sur la page HTML on vient liés toutes nos pages associer :

<!DOCTYPE html>

<html lang="en">    <!-- Langue du site -->

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Titre de la page</title>

  <!--Liens pour des bibliothèque en ligne (Font/Api/Cloud/CDN...)-->
  <link href="https://google.font/css...">

  <!--Liens pour lier les pages externes CSS et JS.-->
  <link rel="stylesheet" href="page.css">
  <script src="script.js" defer></script>
</head>

<body>

  <!-- Ecrire tout le corps dans le body de la page avant de toucher au style-->

</body>
</html>

4. À connaître

  • Shortcut : ! et tab
  • Écrire un commentaire : <!-- le commentaire -->
