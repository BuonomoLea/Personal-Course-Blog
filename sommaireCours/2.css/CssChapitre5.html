Chapitre 5 : Textes et typographies

Choisir une police

On utilise la propriété font-family pour définir la police :

body {
    font-family: Arial, sans-serif;
    }

On peut aussi utiliser Google Fonts pour intégrer des polices externes.

Taille, graisse, style

  • font-size : taille du texte (px, em, rem...)
  • font-weight : graisse (normal, bold, 400, 700...)
  • font-style : italique ou normal

Alignement et hauteur de ligne

  • text-align : left, right, center, justify
  • line-height : espacement vertical entre lignes

Autres propriétés utiles

  • text-transform : uppercase, lowercase, capitalize
  • text-decoration : underline, line-through, none
  • letter-spacing : espacement entre les lettres
  • word-spacing : espacement entre les mots
  • font-style : normal, italic, oblique, inherit, initial, unset

Google Fonts & fonts externes

On importe la police en haut du css avant son utilisation :

@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');
body { 
                font-family: 'Roboto', sans-serif; 
                }
