Chapitre 11 : Caractères typographiques et autres

Entités HTML

Pour afficher certains caractères spéciaux, il faut utiliser des entités HTML comme :

Caractères typographiques courants
Symbole Code HTML Signification
  &nbsp; Espace insécable
< &lt; Signe inférieur (balise ouvrante)
> &gt; Signe supérieur (balise fermante)
& &amp; Esperluette
" &quot; Guillemet double
' &apos; Apostrophe (moins courant)

Bonnes pratiques typographiques

Quelques recommandations pour une typographie web lisible et accessible :

  • Utilisez des guillemets français : &laquo; et &raquo;.
  • Ajoutez une espace insécable avant les deux-points, points d'interrogation et d'exclamation.
  • Utilisez les balises <q> pour les citations courtes, et <blockquote> pour les longues citations.
