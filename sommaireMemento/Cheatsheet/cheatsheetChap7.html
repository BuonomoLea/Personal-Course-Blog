Modèle de boîte & margin / paddings

Modèle de boîte

Le modèle de boîte est composé de : content, padding, border, et margin.

Propriétés clés


        width / height  

        box-sizing : content-box | border-box

box-sizing global

Bonne pratique pour uniformiser le modèle de boîte sur tous les éléments :


        *, *::before, *::after { 
          box-sizing: border-box; 
        }

Paddings (espace intérieur)

Marges (espace extérieur)

Définit l’espace extérieur autour d’un élément :

margin-top
        margin-right
        margin-bottom
        margin-left

Shorthand margin

Déclaration raccourcie pour les marges dans l’ordre : haut, droite, bas, gauche.


        margin: 1rem 0.5rem 1.5rem 0;

Paddings

Définit l’espace intérieur entre le contenu et la bordure de l’élément :


        padding-top
        padding-right
        padding-bottom
        padding-left

Shorthand padding

Déclaration raccourcie pour les paddings dans l’ordre : haut, droite, bas, gauche.


        padding: 0.5rem 1rem; 
        /* haut/bas 0.5 rem, droite/gauche 1 rem */

Tailles de spacing courantes

Unités recommandées

Unités fréquemment utilisées pour margin et padding :


        0
        0.5 rem
        1 rem
        1.5 rem
        2 rem
        3 rem

Exemples d’usage

Cas pratiques d’utilisation des espacements :


        .container { 
          padding: 2rem; 
          margin-bottom: 2rem; 
        }

        .card + .card { 
          margin-top: 1.5rem; 
        }
