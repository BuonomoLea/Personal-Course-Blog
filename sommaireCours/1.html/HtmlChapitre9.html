Chapitre 9 : Navigation entre les pages

1. Navigation entre les pages avec <a href="">

Liens relatifs :


        <a href="page2.html">Aller à la page 2</a>
        <a href="dossier/contact.html">Contact</a>
        <a href="../index.html">Retour à l’accueil</a>

Liens absolus :


        <a href="https://exemple.com">Site externe</a>

2. Navigation sur une même page

a. Créer un point d’ancrage :


        <h2 id="services">Nos services</h2>

b. Faire un lien vers cette ancre :


        <a href="#services">Voir les services</a>

Pour qu’au clic le scroll automatique vers l’élément ayant cet ID soit plus agréable, on applique du CSS :


        * {
        scroll-behavior: smooth;
        }

c. Cibler un élément d’une autre page :


        <a href="page2.html#formulaire">Aller directement au formulaire sur la page 2</a>

3. Arborescence d’un projet web classique


        /mon-site/
        ├── index.html
        ├── /page/
        │   ├── contact.html
        │   ├── services.html
        │   └── mentions-legales.html
        ├── /css/
        │   └── style.css
        ├── /js/
        │   └── script.js
        └── /assets/
            └── logo.png

Conseils :

  • Noms de fichiers et dossiers sans espaces ni majuscules
  • Organiser le contenu dans des sous-dossiers
  • Toujours mettre index.html à la racine
