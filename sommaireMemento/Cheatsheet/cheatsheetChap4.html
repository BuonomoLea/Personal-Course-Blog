Les sélecteurs

Sélecteurs de base


* : tous les éléments

Element : tous les éléments <element>

.class : éléments avec class="class"

#id : élément unique avec id="id"

Combinateurs


parent descendant : tout <descendant> dans <parent>

parent > enfant : chaque enfant direct de <parent>

élément1 + élément2 : <élément2> de même niveau, le premier après <élément1>

élément1 ~ élément2 : tous les <élément2> qui suivent <élément1> au même niveau dans le DOM

Sélecteurs d’attributs


[att] : présence de l’attribut

[att=value] : valeur exacte

[att~=value] : mot dans un espace-séparé

[att|=value] : valeur ou préfixe + tiret

[att^=value] : commence par

[att$=value] : finit par

[att*=value] : contient

Exemple : [href$=".pdf"]  

<li><a href="guide.pdf">Télécharger le guide (PDF)</a></li>
<li><a href="rapport.docx">Télécharger le rapport (DOCX)</a></li>
<li><a href="brochure.pdf">Télécharger la brochure (PDF)</a></li>

Pseudo-classes


États interactifs

:hover
:active
:focus

Structure

:first-child, :last-child
:nth-child(n), :nth-of-type(n)
:not(selector)

Formulaires

:enabled, :disabled, :checked

Pseudo-éléments


::before
::after
::first-line
::first-letter
::selection

Spécificité & ordre


ID > (100)
Classe/attribut/pseudo-classe > (10)
Élément/pseudo-élément > (1)
Dernière règle en cas d’égalité > (cascade)  
!important > (à éviter)

Sélecteurs CSS4


:is(s1,s2,…) > sélectionne si l’élément correspond à l’un des s1, s2…  
:where(s1,s2,…) > même fonctionnement que :is() 
:has(s) > sélectionne l’élément qui contient en descendant un élément correspondant à s 
:not(s1,s2,…) > exclut les éléments correspondant à l’un des s1, s2

Sélecteurs de Container Queries


1. container-type: size | inline-size | block-size → active la surveillance de taille sur un élément 

2. container-name: nom → donne un identifiant pour le cibler dans @container 

@container <nom>? (<condition>) { … } ;  

Conditions : width, min-width, max-width, height, min-height, max-height

Syntaxe nominative :  
@container nom (min-width: 600px) { … } 

Syntaxe globale :  
@container (max-width: 400px) { … }

Sélecteurs de Media Queries


Mobile-first : @media (min-width: valeurPx)
Desktop-first : @media (max-width: valeurPx)

Conditions : width / height / orientation / resolution

Breakpoints usuels : 
xs : 0 – 575px 
sm : ≥ 576px 
md : ≥ 768px 
lg : ≥ 992px 
xl : ≥ 1200px 
xxl : ≥ 1400px 

Exemples :
@media (max-width: 575px) { … }
@media (min-width: 768px) { … }
