Chapitre 12 : Éléments natifs particuliers

Styliser les listes

Les listes HTML peuvent être totalement personnalisées avec des propriétés CSS :

ul {
    list-style-type: square;
    padding-left: 1.5rem;
    }
    li {
    color: #333;
    }

Styliser les tableaux

Les tableaux peuvent être mis en forme pour plus de lisibilité et de style :

table {
    border-collapse: collapse;
    width: 100%;
    }
    th, td {
    border: 1px solid #ccc;
    padding: 0.5rem;
    text-align: left;
    }
    thead {
    background-color: #f0f0f0;
    }

Styliser les formulaires

Les champs de formulaire peuvent varier selon les navigateurs. Il est souvent nécessaire de les uniformiser :

input[type="text"],
    select,
    textarea {
    padding: 0.5rem;
    border: 1px solid #ccc;
    border-radius: 4px;
    width: 100%;
    }
    button {
    background-color: #007BFF;
    color: white;
    padding: 0.5rem 1rem;
    border: none;
    border-radius: 4px;
    cursor: pointer;
    }

Accessibilité et cohérence

Pensez toujours à la lisibilité et à la compatibilité entre navigateurs. Utilisez les sélecteurs adaptés, évitez les styles cachant des éléments natifs comme les select ou les input[type="checkbox"] sans remplacement accessible.

Fonctions CSS utiles et variables

Fonctions CSS

clamp(min, val, max) adapte automatiquement une valeur entre un minimum et un maximum, utile par exemple pour la taille de police :

font-size: clamp(1rem, 2vw, 2rem);

calc() permet d’effectuer des calculs dynamiques dans les propriétés CSS :

width: calc(100% - 2rem);

aspect-ratio contrôle le ratio largeur/hauteur d’un élément :

video { aspect-ratio: 16 / 9; }

Variables CSS / Custom properties

Déclaration de variables globales dans :root :

:root {
  --couleur-primaire: #1a73e8;
  --rayon: 8px;
}

Utilisation dans des sélecteurs :

button {
  background: var(--couleur-primaire);
  border-radius: var(--rayon);
}

Organisation recommandée : placer ces variables dans un fichier variables.css à importer dans tous tes composants.

On peut combiner les variables CSS avec des thèmes ou des animations, notamment grâce à la règle @property.
