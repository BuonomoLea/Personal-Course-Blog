Les balises sémantiques

Structure principale

  • <html> : racine du document
  • <head> : métadonnées, imports, titre
  • <body> : contenu visible

Sectioning content

  • <header> : en-tête de page/section
  • <nav> : menu de navigation
  • <main> : contenu principal unique
  • <section> : bloc thématique avec titre
  • <article> : contenu autonome (post, actu)
  • <aside> : contenu périphérique (sidebar, widget)
  • <footer> : pied de page/section

Grouping content

  • <figure> : image ou média autonome
  • <figcaption> : légende de la figure
  • <details> : bloc repliable (accordéon)
  • <summary> : résumé cliquable de <details>
  • <dialog> : boîte de dialogue native

Text content

  • <h1> – <h6> : titres hiérarchiques
  • <p> : paragraphe
  • <address> : coordonnées de contact
  • <blockquote> : citation longue
  • <q> : citation courte
  • <pre> : texte préformaté
  • <hr> : séparation thématique

Text-level semantics

  • <a> : lien hypertexte
  • <strong> / <b> : importance / gras
  • <em> / <i> : emphase / italique
  • <mark> : surlignage
  • <small> : note ou fine imprimé
  • <abbr> : abréviation
  • <cite> : titre d’œuvre
  • <dfn> : définition
  • <time> : date/heure (attribut datetime)
  • <del> / <ins> : suppression / insertion
  • <sub> / <sup> : indice / exposant
  • <bdi> : isolation directionnelle
  • <bdo> : override directionnel

Embedded content

  • <img> : image
  • <picture> + <source> : images responsive
  • <audio> / <video> + <source> + <track>
  • <embed> / <object> : contenu tiers
  • <canvas> : dessin dynamique
  • <svg> : vecteurs

Tabular data

  • <table> : tableau
  • <caption> : titre de tableau
  • <thead> / <tbody> / <tfoot>
  • <tr> / <th> / <td>
  • <colgroup> / <col>

Forms & inputs

  • <form> : conteneur de champs
  • <fieldset> / <legend>
  • <label> : étiquette associée
  • <input> (types : text, email, checkbox…)
  • <textarea> / <button> / <select>
  • <optgroup> / <option>
  • <datalist> / <output>
  • <progress> / <meter>

Interactive elements

  • <details> / <summary> (déjà listé)
  • <dialog> (déjà listé)
  • <menu> / <menuitem> (menus contextuels)
  • <slot> (Web Components)

Balises non-sémantiques

  • <div> : conteneur bloc générique
  • Alternatives sémantiques : <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>
  • <span> : conteneur inline générique
  • Alternatives sémantiques : <strong>, <em>, <mark>, <code>, <time>, <abbr>
