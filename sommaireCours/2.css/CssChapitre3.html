Chapitre 3 : Les bases du CSS

Structure d'une règle CSS

Une règle CSS est composée d’un sélecteur, d’une ou plusieurs propriétés, et de leurs valeurs :

sélecteur {
    propriété: valeur;
}

Exemple :

p {
    color: red;
    font-size: 16px;
}

Propriétés CSS courantes

  • color : couleur du texte
  • background-color : couleur de fond
  • font-size : taille de la police
  • margin : marge extérieure
  • padding : marge intérieure
  • border : bordure

Valeurs et unités

Les valeurs CSS peuvent utiliser différentes unités :

Unités CSS courantes
Unité Usage
pxValeur fixe (pixels)
%Pourcentage de l’élément parent
emRelative à la taille de police de l’élément
remRelative à la taille de police de la racine
vh / vw1% de la hauteur / largeur de la fenêtre

Commentaires en CSS

Les commentaires permettent d’expliquer le code :

/* Ceci est un commentaire CSS */

Spécificité & Cascade

La spécificité détermine quelle règle CSS s’applique si plusieurs ciblent le même élément :

  • Inline styles : 1000 points
  • Sélecteurs ID (#id) : 100 points
  • Classes, pseudo-classes, attributs : 10 points
  • Types, pseudo-éléments : 1 point

L’ordre d’application des styles est :

  1. Styles du navigateur (user agent)
  2. Fichiers externes et <style> (auteurs)
  3. Inline styles
  4. !important (force la priorité)

En cas d’égalité, la dernière règle dans le code l’emporte.

.alert {
    color: red !important;
}
