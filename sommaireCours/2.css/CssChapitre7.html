Chapitre 7 : Bordures et ombres

Définir des bordures en CSS

Les bordures peuvent être définies avec les propriétés suivantes :

  • border-width : épaisseur de la bordure
  • border-style : style de la bordure (solid, dashed, dotted...)
  • border-color : couleur de la bordure
.element {
    border: 2px solid #333;
    }

Arrondir les coins avec border-radius

Pour créer des coins arrondis :

.element {
    border-radius: 10px;
    }

Appliquer des ombres avec box-shadow

Les ombres permettent d’ajouter de la profondeur aux éléments :

.element {
    box-shadow: 4px 4px 10px rgba(0, 0, 0, 0.3);
    }
  • Premier paramètre : décalage horizontal
  • Deuxième : décalage vertical
  • Troisième : flou
  • Quatrième : couleur (souvent en rgba)

Ombres multiples

Vous pouvez empiler plusieurs ombres :

.element {
    box-shadow: 
        2px 2px 5px rgba(0, 0, 0, 0.2),
        -2px -2px 5px rgba(255, 255, 255, 0.2);
    }
