Chapitre 10 : Éléments interactifs natifs

Introduction aux éléments interactifs

Certains éléments HTML intègrent nativement des comportements interactifs sans besoin de JavaScript.

Les éléments <details> et <summary>

Ces éléments créent un petit bloc repliable, parfait pour des FAQ, infos secondaires, etc.

Exemple :


        <details>
        <summary>Afficher la réponse</summary>
        <p>Voici la réponse à la question posée…</p>
        </details>

Comportement :

  • Cliquer sur <summary> affiche ou cache le reste du contenu.
  • Ouvert par défaut avec l’attribut open : <details open>

<dialog>

Crée un popup sans dépendance JavaScript lourde.

Exemple de base :


        <dialog id="modal">
        <p>Bonjour</p>
        <button onclick="document.getElementById('modal').close()">Fermer</button>
        </dialog>

        <button onclick="document.getElementById('modal').showModal()">Ouvrir le dialogue</button>

À savoir :

  • Méthodes JavaScript : .showModal() (modal) ou .show()
  • Utilisation de l’attribut open pour cibler les styles

<progress> et <meter>

Pour représenter graphiquement une valeur ou un état.

<progress> : Barre de progression

Montre une progression vers un objectif (ex : upload, téléchargement)


        <progress value="60" max="100">60%</progress>

<meter> : Jauge de valeur

Affiche une valeur dans une plage (ex : note, niveau, température…)


        <meter value="0.7" min="0" max="1">70%</meter>

Attributs utiles : low, high, optimum pour styliser dynamiquement la jauge.
