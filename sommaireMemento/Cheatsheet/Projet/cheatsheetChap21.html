Terminal :

Commandes Git

Indispensables pour gérer les projets versionnés.

git clone URLDuDépôt : Clone un dépôt Git distant vers votre machine locale.

Exemple : git clone https://github.com/mon-depot.git.

git status : Affiche l’état des modifications dans le dépôt.

Exemple : git status.

Utilité : Savoir quels fichiers sont modifiés, ajoutés ou en attente de commit.

git add NomDuFichier : Ajoute un fichier spécifique pour le prochain commit.

Exemple : git add fichier.txt.

Utilité : Préparer vos changements pour les enregistrer.

git commit -m "Message" : Sauvegarde les modifications ajoutées avec un message descriptif.

Exemple : git commit -m "Ajout de nouvelles fonctionnalités".

git push : Envoie vos commits locaux vers le dépôt distant.

Exemple : git push origin main.

git pull : Récupère les dernières modifications depuis le dépôt distant.

Exemple : git pull origin main.

git log : Affiche l’historique des commits.

Exemple : git log.

git checkout -b NomDeBranche : Crée une nouvelle branche et bascule dessus.

Exemple : git checkout -b feature-ajout.

Utilité : Travailler sur des fonctionnalités séparées.

git merge NomDeBranche : Fusionne une branche spécifique dans la branche courante.

Exemple : git merge feature-ajout
