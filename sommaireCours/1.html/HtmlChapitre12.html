Chapitre 12 : Résumé à appliquer sur chaque projet

Structure HTML de base

  • Utiliser la déclaration <!DOCTYPE html>
  • Ajouter les balises <html lang="fr">, <head>, <body>
  • Inclure les balises <meta charset>, <meta viewport>, <title>, et <meta description>

Sémantique et accessibilité

  • Utiliser les balises sémantiques : <header>, <main>, <nav>, <section>, <article>, <footer>
  • Ajouter des attributs ARIA comme aria-label si nécessaire
  • Structurer avec des titres hiérarchiques <h1> à <h6>

Bonnes pratiques CSS associées

  • Lier un fichier de variables CSS (variables.css)
  • Séparer le style général (style.css) du layout spécifique (layoutSummary.css)
  • Utiliser des classes réutilisables et nommées clairement (BEM, utilitaires, etc.)

Footer et scripts

  • Inclure le <footer> avec des infos légales
  • Placer les scripts JavaScript en bas de page avec defer
