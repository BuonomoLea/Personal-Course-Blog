Chapitre 10 : Responsive design

Qu’est-ce que le responsive design ?

Le responsive design consiste à adapter une page web à toutes les tailles d’écran : ordinateurs, tablettes, smartphones, etc.

Objectif : offrir une expérience utilisateur optimale, quelle que soit la résolution.

Utiliser les media queries

Les @media queries permettent d’appliquer des styles en fonction des caractéristiques du périphérique.

@media screen and (max-width: 768px) {
    .menu {
        flex-direction: column;
    }
    }

Unités relatives et flexibilité

  • % : basé sur l’élément parent
  • em, rem : basé sur la taille de police
  • vw, vh : basé sur la taille de la fenêtre

L’utilisation de ces unités permet une mise en page plus fluide.

Bonnes pratiques

  • Utiliser une grille flexible (Flexbox ou Grid)
  • Prévoir une navigation mobile claire
  • Éviter les largeurs fixes
  • Tester sur plusieurs tailles d’écrans
