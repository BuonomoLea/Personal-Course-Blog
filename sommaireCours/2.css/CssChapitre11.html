Chapitre 11 : Transitions et animations

Les transitions CSS

Les transitions permettent d’ajouter des effets de changement progressif entre deux états CSS (par exemple, au survol d’un bouton).

button {
    background-color: blue;
    transition: background-color 0.3s ease;
    }
    button:hover {
    background-color: green;
    }

Les animations CSS (keyframes)

Les animations créent des effets complexes en plusieurs étapes définies par des @keyframes.

@keyframes slideIn {
    from {
        transform: translateX(-100%);
    }
    to {
        transform: translateX(0);
    }
    }

    .element {
    animation: slideIn 1s ease-in-out;
    }

Bonnes pratiques

  • Ne pas surcharger une page avec trop d’animations
  • Respecter l’accessibilité (éviter les mouvements excessifs)
  • Utiliser prefers-reduced-motion si nécessaire
@media (prefers-reduced-motion: reduce) {
    * {
        animation: none !important;
        transition: none !important;
    }
    }
