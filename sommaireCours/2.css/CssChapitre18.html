Chapitre 18 : Style de menu de navigation

Pourquoi styliser le menu de navigation ?

Le menu de navigation est un élément clé pour orienter les utilisateurs. Un bon style améliore la lisibilité, l’ergonomie et la cohérence du site.

Techniques de style courantes

Structure de base HTML

<nav role="navigation" aria-label="Menu principal">
  <ul class="navbar">
    <li><a href="#">Accueil</a></li>
    <li><a href="#services">Services</a></li>
    <li><a href="#contact">Contact</a></li>
  </ul>
</nav>

Style horizontal

.navbar {
  display: flex;
  gap: 2rem;
  list-style: none;
  padding: 1rem 2rem;
  background: #fff;
}
.navbar a {
  text-decoration: none;
  color: var(--text-color);
  font-weight: 500;
}
.navbar a:hover {
  color: var(--accent-color);
}

Sticky en haut

nav {
  position: sticky;
  top: 0;
  z-index: 100;
  background-color: white;
  box-shadow: 0 2px 8px rgba(0,0,0,0.05);
}

Responsive avec menu mobile

Sur mobile, afficher le menu en colonne, caché par défaut, avec un bouton pour l’afficher :

@media (max-width: 768px) {
  .navbar {
    flex-direction: column;
    display: none;
  }
  .navbar.open {
    display: flex;
  }
}
/* Bouton : <button aria-expanded="false" aria-controls="menu">☰</button> */
/* JS léger pour toggle .open sur .navbar et mise à jour aria-expanded */

Accessibilité des menus

  • <nav role="navigation" aria-label="Menu principal"> pour la sémantique
  • Lien actif : utiliser aria-current="page" sur l’élément courant
  • Navigation clavier fluide, éviter pièges de focus
  • Bouton mobile avec aria-controls et aria-expanded pour informer du menu

Sidebar de navigation complète (exemple avancé)

<nav>
  <ul>
    <li class="home">
      <a href="">
        <div><svg></svg></div>
        <span>Home</span>
      </a>
    </li>
    <!-- autres items similaires -->
  </ul>
</nav>

Extrait CSS pour sidebar

a {
  display: flex;
  align-items: center;
  text-decoration: none;
  overflow: hidden;
  border-top: 2px solid rgba(237,237,237,.05);
}
a:hover {
  background: #262626;
}
nav {
  grid-area: nav;
  display: flex;
  background-color: #212121;
}
li {
  list-style-type: none;
}
div {
  padding: 20px;
  width: 65px;
}
svg {
  width: 80%;
  margin: 0 auto;
  display: block;
}
span {
  font-size: 1.5rem;
  color: #E9F8FF;
  width: 0;
  transition: width 400ms ease, margin-left 300ms ease;
}
nav:hover span {
  width: 130px;
  margin-left: 30px;
}

Accessibilité

Assurez-vous que le menu est accessible au clavier, que le contraste est suffisant et que les informations ARIA sont correctement utilisées.
