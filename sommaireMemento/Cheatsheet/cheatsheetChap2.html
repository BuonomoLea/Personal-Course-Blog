Les classes et ID génériques :

Layout & display

  • .d-block → display: block;
  • .d-inline → display: inline;
  • .d-none → display: none;
  • .d-flex → display: flex;
  • .flex-row → flex-direction: row;
  • .flex-col → flex-direction: column;
  • .clearfix → clear: both;

Spacing (m = margin, p = padding)

  • .m-0 → margin: 0;
  • .mt-1 → margin-top: .25rem;
  • .mb-2 → margin-bottom: .5rem;
  • .mx-auto → margin-left: auto; margin-right: auto;
  • .p-0 → padding: 0;
  • .pt-1 → padding-top: .25rem;
  • .py-2 → padding: .5rem;
  • .px-3 → padding: .75rem;

Typography

  • .text-left → text-align: left;
  • .text-center → text-align: center;
  • .text-right → text-align: right;
  • .text-muted → color: #6c757d;
  • .text-primary → color: #007bff;
  • .text-danger → color: #dc3545;
  • .font-bold → font-weight: 700;
  • .font-italic → font-style: italic;

Sizing & helpers

  • .w-100 → width: 100%;
  • .h-100 → height: 100%;
  • .max-w-xxl → max-width: 1140px;
  • .visually-hidden → position: absolute; width: 1px; height: 1px; padding: 0; margin: -1px; overflow: hidden; clip: rect(0,0,0,0); white-space: nowrap; border: 0;
  • .overflow-auto → overflow: auto;

Components scratch

  • .btn → display: inline-block; padding: .5rem 1rem; border: 1px solid transparent; border-radius: .25rem;
  • .btn-primary → background-color: #007bff; color: #fff;
  • .btn-secondary → background-color: #6c757d; color: #fff;
  • .card → border: 1px solid #dee2e6; border-radius: .25rem;
  • .card-header → padding: .75rem 1.25rem; background-color: #f7f7f7;
  • .card-body → padding: 1.25rem;

IDs structurants

  • #header → header principal
  • #main → conteneur principal
  • #main-content → contenu principal
  • #footer → pied de page
  • #nav → nav primaire
  • #sidebar → barre latérale
  • #logo → wrapper logo
  • #search → conteneur recherche
  • #modal → modale
  • #alert → message alerte
  • #cookie-consent → bandeau cookies
  • #back-to-top → bouton retour haut
  • #skip-nav → lien skip-nav
