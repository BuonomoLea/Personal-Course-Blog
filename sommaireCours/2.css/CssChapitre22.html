Chapitre 22 : Effets visuels et micro-interactions

Effets visuels en CSS

1. Surlignement animé

Mettre en évidence un texte par un effet de surlignage qui se déploie progressivement au survol ou focus.

Structure HTML :

<p class="highlight">Texte à surligner</p>

Mise en forme CSS :

.highlight {
                position: relative;
                display: inline-block;
                color: #111;
                }
                .highlight::after {
                content: '';
                position: absolute;
                left: 0; bottom: 0.1em;
                width: 100%; height: 0.3em;
                background: #ffeb3b;
                transform: scaleX(0);
                transform-origin: left;
                transition: transform 0.4s ease;
                z-index: -1;
                }
                .highlight:hover::after,
                .highlight:focus::after {
                transform: scaleX(1);
                }

Micro-interactions

2. Loader CSS

Afficher un élément tournant pendant le chargement de données.

<div class="loader"></div>
.loader {
                width: 48px; height: 48px;
                border: 6px solid #ddd;
                border-top-color: #3f51b5;
                border-radius: 50%;
                animation: spin 1s linear infinite;
                margin: 2rem auto;
                }
                @keyframes spin {
                to { transform: rotate(360deg); }
                }

3. Barre de progression au scroll

Afficher une barre dont la largeur reflète la progression de lecture dans la page.

<div class="scroll-progress"></div>
.scroll-progress {
                position: fixed;
                top: 0; left: 0;
                height: 4px;
                background: #3f51b5;
                width: 0;
                z-index: 1000;
                transition: width 0.1s ease-out;
                }

Comportement JavaScript :

const progress = document.querySelector('.scroll-progress');
                window.addEventListener('scroll', () => {
                const docHeight = document.documentElement.scrollHeight - window.innerHeight;
                const scrolled  = window.scrollY;
                const pct       = (scrolled / docHeight) * 100;
                progress.style.width = pct + '%';
                });

Exemples pratiques

4. Skeleton loaders

Simuler la structure d’un contenu pendant son chargement.

<div class="card-skeleton">
                <div class="avatar-skel"></div>
                <div class="text-skel title"></div>
                <div class="text-skel line"></div>
                <div class="text-skel line short"></div>
                </div>
.card-skeleton {
                width: 300px;
                padding: 1rem;
                border: 1px solid #ddd;
                border-radius: 6px;
                background: #fff;
                }
                .avatar-skel {
                width: 60px; height: 60px;
                border-radius: 50%;
                background: #eee;
                margin-bottom: 1rem;
                }
                .text-skel {
                height: 12px;
                background: #eee;
                margin-bottom: .8rem;
                border-radius: 4px;
                }
                .text-skel.short { width: 40%; }
                .text-skel.title { width: 70%; height: 16px; }

                @keyframes pulse {
                0%,100% { opacity: 1; }
                50%     { opacity: .4; }
                }
                .avatar-skel,
                .text-skel {
                animation: pulse 1.5s ease-in-out infinite;
                }

5. Hover créatif sur images et textes

Zoom d’image et fondu :

<div class="img-zoom">
                <img src="photo.jpg" alt="Description">
                </div>
.img-zoom {
                overflow: hidden;
                display: inline-block;
                }
                .img-zoom img {
                display: block;
                transition: transform .5s ease, filter .5s ease;
                }
                .img-zoom:hover img,
                .img-zoom:focus img {
                transform: scale(1.1);
                filter: brightness(1.1);
                }

Soulignement animé de titre :

<h2 class="underline">Titre interactif</h2>
.underline {
                position: relative;
                display: inline-block;
                color: #111;
                }
                .underline::after {
                content: '';
                position: absolute;
                left: 0; bottom: -0.1em;
                width: 100%; height: 2px;
                background: #3f51b5;
                transform: scaleX(0);
                transform-origin: right;
                transition: transform .3s ease;
                }
                .underline:hover::after,
                .underline:focus::after {
                transform: scaleX(1);
                transform-origin: left;
                }
