Unités relatives et valeurs fluides

Unités relatives : % – em – rem – vw / vh – vmin / vmax

Pourcentage (%)

Référence à la taille de l’élément parent Exemple : 


            .container { 
            width: 90%;         /* 90% de la largeur du parent */ 
            margin: 0 auto;     /* centré horizontalement */ 
            }

em

Référence à la taille de police de l’élément parent Exemple :


            body { 
            font-size: 16px; 
            } 
            h1 { 
            font-size: 2em;     /* équivalent à 32px */ 
            } 
            button { 
            padding: 0.75em 1.5em; /* 12px 24px */ 
            }

rem

Référence à la taille de police de la racine (html) Exemple :


            html { 
            font-size: 16px; 
            } 
            p { 
            margin-bottom: 1.5rem; /* 24px */ 
            }

vw / vh

1 vw = 1 % de la largeur du viewport 1 vh = 1 % de la hauteur du viewport Exemple :


            .hero { 
            height: 100vh;      /* 100% de la hauteur du viewport */ 
            } 
            .title { 
            font-size: 5vw;     /* 5% de la largeur du viewport */ 
            }

vmin / vmax

1 vmin = plus petite valeur entre vw et vh 1 vmax = plus grande valeur entre vw et vh Exemple :


            .box { 
            width: 40vmin;      /* 40% de la plus petite des dimensions viewport */ 
            height: 40vmin; 
            }

Valeurs fluides : clamp() – min() / max() – calc()

clamp()

Fixe une valeur fluide entre un minimum et un maximum Syntaxe : clamp(min, valeur fluide, max) Exemple :


        h2 { 
        font-size: clamp(1.25rem, 4vw, 2rem); 
        /* min 1.25rem – fluide 4vw – max 2rem */ 
        }

min() / max()

Choisit respectivement la plus petite ou la plus grande valeur parmi les arguments Exemple :


        .card { 
        width: min(30rem, 90vw); 
        /* au plus 30rem sinon 90% du viewport */ 
        } 
        .sidebar { 
        width: max(20rem, 25%); 
        /* au moins 20rem sinon 25% du parent */ 
        }

calc()

Effectue des calculs CSS combinant unités différentes Exemple :


        .page { 
        width: calc(100% - 2rem); 
        /* 100% du parent moins 32px */ 
        }

env()

Accède aux variables d’environnement (safe-area-inset pour les “notch”) Exemple :


        body { 
        padding: env(safe-area-inset-top) 
                env(safe-area-inset-right) 
                env(safe-area-inset-bottom) 
                env(safe-area-inset-left); 
        }
