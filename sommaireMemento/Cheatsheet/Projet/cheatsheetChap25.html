Terminal :

Gestion d’archives

Utile pour compresser ou extraire des fichiers.

tar -czvf archive.tar.gz NomDuFichier : Crée une archive compressée contenant NomDuFichier.

Exemple : tar -czvf mon_projet.tar.gz dossier/.

Utilité : Regrouper plusieurs fichiers et réduire leur taille pour les partager.

tar -xzvf archive.tar.gz : Décompresse une archive .tar.gz.

Exemple : tar -xzvf mon_projet.tar.gz.

unzip fichier.zip : Décompresse un fichier ZIP.

Exemple : unzip fichiers.zip.
