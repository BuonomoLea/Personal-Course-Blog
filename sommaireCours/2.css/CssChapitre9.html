Chapitre 9 : Mise en page moderne

Flexbox

Flexbox est un modèle de mise en page unidimensionnelle. Il est parfait pour aligner des éléments horizontalement ou verticalement.

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
  • justify-content : aligne les éléments sur l’axe principal
  • align-items : aligne les éléments sur l’axe secondaire
  • flex-direction : définit l’orientation (ligne ou colonne)

CSS Grid

Grid est un système de mise en page bidimensionnel : colonnes et lignes.

.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr;
  gap: 20px;
}
  • grid-template-columns : définit les colonnes
  • gap : espace entre les éléments
  • grid-template-areas : organisation visuelle nommée

Techniques responsive

Les mises en page modernes doivent s’adapter aux écrans variés :

  • Médias queries (@media)
  • Unités relatives : em, rem, %, vw, vh
  • Propriétés adaptatives : min-width, max-width, etc.
