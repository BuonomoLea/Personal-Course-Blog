Chapitre 4 : Couleurs et arrière-plans

Types de couleurs en CSS

Différents formats existent pour définir une couleur :

Formats de couleur CSS
Format Exemple Détails
Hexadécimal#ff6600Très utilisé (valeur RRGGBB)
RGBrgb(255, 100, 50)Valeurs de 0 à 255
RGBArgba(255, 100, 50, 0.5)Avec transparence (0 à 1)
HSLhsl(30, 100%, 50%)Teinte, saturation, luminosité
HSLAhsla(30, 100%, 50%, 0.3)Ajoute l’opacité

Propriétés liées aux couleurs

  • color : couleur du texte
  • background-color : couleur de fond
  • border-color : couleur de la bordure

Images de fond

Exemple d’utilisation d’une image en fond avec les propriétés associées :

body {
    background-image: url('/assets/img/fond.jpg');
    background-repeat: no-repeat;
    background-size: cover;
    background-position: center;
}

Dégradés

On peut créer des dégradés linéaires ou radiaux :

div {
    background: linear-gradient(to right, #ff6600, #ffcc00);
}

Les directions possibles sont to right, to bottom, etc., et plusieurs couleurs ou transparences peuvent être utilisées.
