Chapitre 8 : Positionnement et affichage

La propriété display

La propriété display détermine le mode d'affichage d’un élément.

  • block : occupe toute la largeur disponible
  • inline : s’aligne dans une ligne sans retour à la ligne
  • inline-block : comme inline, mais accepte les dimensions
  • flex : conteneur flexible pour organiser ses enfants
  • grid : conteneur en grille
  • none : rend l’élément invisible et non pris en compte dans le flux

La propriété position

Permet de positionner les éléments avec précision :

  • static (par défaut)
  • relative : positionné par rapport à sa position initiale
  • absolute : positionné par rapport à son premier ancêtre positionné
  • fixed : position fixe par rapport à la fenêtre
  • sticky : reste fixe dans une certaine zone lors du scroll
.element {
    position: absolute;
    top: 10px;
    left: 20px;
    }

La propriété z-index

Contrôle la superposition des éléments positionnés :

.element {
    position: relative;
    z-index: 10;
    }

Float et Clear

float permet de faire "flotter" un élément à gauche ou à droite :

.image {
    float: right;
    }

Pour éviter que les éléments suivants ne l’enveloppent :

.suivant {
    clear: both;
    }
