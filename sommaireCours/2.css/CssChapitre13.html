Chapitre 13 : Compatibilité navigateurs et préfixes

Pourquoi gérer la compatibilité ?

Les différents navigateurs ne supportent pas toujours les mêmes propriétés CSS de la même façon. Pour assurer un rendu homogène, il est important de gérer la compatibilité.

Les préfixes CSS

Les préfixes sont des extensions spécifiques à un navigateur qui permettent d’utiliser des fonctionnalités CSS encore expérimentales :

  • -webkit- : Chrome, Safari, Opera
  • -moz- : Firefox
  • -ms- : Internet Explorer et Edge (ancien)
  • -o- : Opera ancien

Exemple :

button {
    -webkit-border-radius: 10px;
    -moz-border-radius: 10px;
    border-radius: 10px;
    }

Autres bonnes pratiques

  • Utiliser des outils comme Autoprefixer pour automatiser la gestion des préfixes.
  • Tester les sites sur plusieurs navigateurs et appareils.
  • Consulter les sites comme Can I Use pour vérifier la compatibilité des propriétés CSS.

Stratégies pour un rendu homogène

Reset CSS

Remise à zéro des marges, paddings et gestion du modèle de boîte :

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

Ou utiliser une librairie comme normalize.css pour harmoniser les styles par défaut entre navigateurs.

Bonnes pratiques CSS

  • Utiliser des unités fluides (rem, %, vw) plutôt que des pixels fixes.
  • Préférer flexbox ou grid pour gérer les tailles et dispositions adaptatives.
  • Éviter les hacks ou un usage excessif de position: absolute.
  • Tester sur plusieurs navigateurs : Chrome, Firefox, Safari, Edge, ainsi que différents systèmes (Windows, macOS).
  • Utiliser des outils comme BrowserStack, Responsively, ou les devtools mobiles pour faciliter les tests.

Design System

  • Centraliser les variables CSS pour une maintenance facilitée.
  • Créer des classes CSS réutilisables et nommées proprement (conventions comme BEM).
  • Tester la responsivité et la compatibilité dès le début du projet, avec une approche mobile-first et l’usage des media queries.
  • Utiliser les outils de développement (F12) pour déboguer et inspecter le rendu.
