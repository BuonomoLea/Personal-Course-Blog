Flexbox & CSS Grid

Flexbox

Liste codes

Principales propriétés du modèle Flexbox :


        display: flex 

        flex-direction : row | row-reverse | column | column-reverse 

        flex-wrap : nowrap | wrap | wrap-reverse 

        justify-content : flex-start | center | flex-end | space-between | space-around | space-evenly 

        align-items : stretch | flex-start | center | flex-end | baseline 

        align-content : stretch | flex-start | center | flex-end | space-between | space-around 

        gap : <longueur> (espacement entre les items)

Propriétés des items

Utilisées pour contrôler le comportement individuel des éléments enfants dans le flex container :


        order : <entier> (ordre de placement)

        flex-grow : <nombre> (facteur d’expansion)

        flex-shrink : <nombre> (facteur de rétrécissement)

        flex-basis : <longueur> (taille initiale)

        flex (shorthand) : <flex-grow> <flex-shrink>? <flex-basis>?

        align-self : auto | flex-start | center | flex-end | baseline | stretch

Exemple – centrer un contenu

Utilisation classique de Flexbox pour centrer un élément horizontalement et verticalement :


        .container { 
          display: flex; 
          justify-content: center; 
          align-items: center; 
        }

CSS Grid

Liste codes

Principales propriétés du modèle Grid pour définir lignes, colonnes et zones :


        display : grid | inline-grid

        grid-template-columns : <track-list> (ex : repeat(3, 1fr) ou 200px 1fr 100px)

        grid-template-rows : <track-list>

        grid-template-areas : "header header" "sidebar main" "footer footer"

        gap : <longueur> (espace entre lignes et colonnes)

        justify-items : start | end | center | stretch

        align-items : start | end | center | stretch

        justify-content : start | end | center | space-between | space-around | stretch

        align-content : start | end | center | space-between | space-around | stretch

        grid-auto-flow : row | column | dense | row dense | column dense

Placement des items

Définir la position des éléments enfants dans la grille :


        grid-column : <début> / <fin>

        grid-row : <début> / <fin>

        grid-area : <nom-zone> | <row-start> / <col-start> / <row-end> / <col-end>

Exemple – grille fluide auto-ajustable

Exemple d’une grille responsive avec colonnes flexibles :


        .container { 
          display: grid; 
          grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); 
          gap: 1rem; 
        }
